## Text Formatting

**This text is bold**

*This text is italic*

***This text is bold and italic***

~~This text is strikethrough~~

You can also use __underscores__ for **bold** and _single underscores_ for *italic*.

## Lists

### Unordered Lists

- Item 1
- Item 2
- Item 3
  - Nested item 3.1
  - Nested item 3.2
    - Deeply nested item 3.2.1
- Item 4

### Ordered Lists

1. First item
2. Second item
3. Third item
   1. Nested item 3.1
   2. Nested item 3.2
4. Fourth item

### Task Lists

- [x] Completed task
- [ ] Uncompleted task
- [ ] Another uncompleted task

## Code

### Inline Code

Use `inline code` by wrapping text in backticks.

You can install packages with `npm install` or `pip install`.

### Code Blocks

```javascript
// JavaScript example
function greet(name) {
  console.log(`Hello, ${name}!`);
}

greet("World");
```

```python
# Python example
def greet(name):
    print(f"Hello, {name}!")

greet("World")
```

```bash
# Bash example
echo "Hello, World!"
```

## Blockquotes

>> This is a nested blockquote.
